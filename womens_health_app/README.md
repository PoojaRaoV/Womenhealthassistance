# Women Health Assistance System 

Project Overview

This project is an AI-driven personalized healthcare system designed to support women's health and wellness. It provides features like diet planning, doctor appointment assistance, and health monitoring.

---

##  Features

* Diet Planner
* Doctor Appointment System
* Find Doctors Nearby
* AI-based Health Recommendations
* User-friendly Interface

---

##  Technologies Used

* Frontend: HTML, CSS,  React
* Backend: Python,Node.js
* Database: MongoDB

---

## ▶️ How to Run the Project

### If using React:

```bash
npm install
npm start
```

Then open:
http://localhost:3000
